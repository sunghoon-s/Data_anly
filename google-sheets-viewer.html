Google Sheets API를 사용하여 데이터를 실시간으로 불러옵니다

🔑 API 설정

💡 사용 방법

  1. Google Cloud Console에서 프로젝트 생성 및 Google Sheets API 활성화
  2. API 키 생성 (사용자 인증 정보 → API 키)
  3. 구글 시트를 "링크가 있는 모든 사용자" 읽기 권한으로 공유
  4. 시트 ID는 URL에서 추출: docs.google.com/spreadsheets/d/[SHEET_ID]/edit
  5. 위 정보를 입력하고 "데이터 불러오기" 클릭
