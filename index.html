Vercel 환경변수를 통한 안전한 구글 시트 데이터 관리

📊 Vercel API 방식 (환경변수 자동 설정)

🔐 환경변수 자동 설정

API 키와 시트 ID는 Vercel 환경변수에서 자동으로 가져옵니다.

⚡ Google Apps Script 방식

🛠️ 설정 가이드

1️⃣ Vercel 환경변수 설정

  1. Google Cloud Console에서 Google Sheets API 키 발급
  2. 구글 시트를 "링크가 있는 모든 사용자" 읽기 권한으로 공유
  3. 구글 시트 URL에서 시트 ID 추출
  4. Vercel 대시보드 → 프로젝트 → Settings → Environment Variables
  5. GOOGLE_API_KEY: 발급받은 API 키 입력
  6. GOOGLE_SHEET_ID: 추출한 시트 ID 입력
  7. Production, Preview, Development 모든 환경에 설정

2️⃣ Google Apps Script 설정 (선택사항)

  1. script.google.com 접속
  2. 새 프로젝트 생성
  3. 아래 코드를 붙여넣기
  4. 배포 → 새 배포 → 유형: 웹앱
  5. 실행 권한: 나, 액세스 권한: 모든 사용자
  6. 생성된 웹앱 URL 복사

💡 구글 시트 ID 찾는 방법

구글 시트 URL: https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
[SHEET_ID] 부분이 필요한 시트 ID입니다.
